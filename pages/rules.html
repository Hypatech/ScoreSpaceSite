RULE 1
All games must include a leaderboard system. This allows players to compete for the highscore in the second half of the jam.

RULE 2
You can make the game in a team of any size. That can be anywhere from just one person all the way to the population of earth. It's recommended to keep the team size small, however - sizes too big become unmanageable fast.

RULE 3
Any game engine or framework is allowed. Simple.

RULE 4
If your game is NSFW, it is required that you have a warning at the start of the game or in the description text. We dont want to be scarred for life, thanks.

RULE 5
You are allowed to use assets made by someone else - ensure that you have full permission and credit is given in the description. However, we strongly advise against this.

RULE 6
Every game submitted must be in a digital format which can be played on the web, Windows, Linux or Mac.
